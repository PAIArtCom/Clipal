## v0.21.3

### English

#### Highlights

- Synced Codex OAuth request wrapping with Codex `0.144.1`.
- Synced Claude Code OAuth request wrapping with Claude Code `2.1.207`.

#### What Changed

- Updated Codex OAuth proxy and refresh-request identities to use the Codex `0.144.1` version and user-agent prefix.
- Updated Claude Code OAuth proxy, usage, and live-smoke identities to use Claude Code `2.1.207`, including its derived billing version.
- Added regression checks for the Codex wrapper identity while retaining the existing Codex TOML and Claude JSON takeover coverage.

#### Release Notes

- No YAML migration is required.
- Existing OAuth credentials continue to work.
