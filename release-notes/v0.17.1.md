## v0.17.1

### English

#### Highlights

- Added npm distribution for Clipal, including both `clipal` and `@paiart/clipal`.
- Updated release automation so pushing a `v*` tag now publishes GitHub Release assets and npm packages in one flow.
- Simplified onboarding docs to recommend `npm install -g clipal` and Web UI based setup first.

#### What Changed

- Added: npm package layouts under `packaging/npm` and `packaging/npm-paiart`.
- Added: postinstall download flow that installs the correct platform binary from GitHub Releases.
- Added: release workflow steps to sync npm package versions from the Git tag and publish both npm packages automatically.
- Improved: README and Getting Started docs now prioritize npm installation and Web UI configuration instead of manual YAML bootstrap.

#### Thanks

- Thanks to the live release rehearsal that helped validate the full GitHub Release plus npm publishing pipeline.
