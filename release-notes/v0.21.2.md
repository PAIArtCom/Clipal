## v0.21.2

### English

#### Highlights

- Synced Claude Code OAuth request wrapping with Claude Code `2.1.201`.
- Fixed false incomplete-response warnings for compressed Claude OAuth streaming responses.

#### What Changed

- Updated Claude Code OAuth wrapper metadata, including user agent, Stainless timeout/runtime version, accepted encodings, and billing fingerprint version.
- Centralized Claude Code request-wrapper metadata under `internal/proxy`, matching the existing Codex OAuth wrapper structure.
- Fixed compressed SSE completion handling so gzip-encoded Claude streams are not scanned as plaintext protocol markers.
- Updated the live Claude OAuth smoke script to request compressed responses with `curl --compressed` and generate `2.1.201` billing payloads.
- Updated Claude OAuth usage requests to use the current Claude Code CLI user agent.

#### Release Notes

- No YAML migration is required.
- Existing OAuth credentials continue to work.
- Compressed Claude OAuth streaming responses should no longer produce misleading `Incomplete response` warnings when the client receives a complete response.

#### 发布说明
